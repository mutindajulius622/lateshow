# Coffee Shop Domain Model

This project models a Coffee Shop domain using object-oriented programming principles in Python.

## Setup and Preparation

1. Create a virtual environment using pipenv:

```bash
pipenv install
pipenv shell
```

2. Install pytest for testing:

```bash
pipenv install pytest
```

## Project Structure

- `customer.py` - Customer class
- `coffee.py` - Coffee class
- `order.py` - Order class
- `tests/` - directory containing test files for each class
- `debug.py` - interactive script to manually test the classes

## Running Tests

Use the command:

```bash
pytest
```

to run all test cases.
