# Bot Battlr

A React application for building your custom bot army.

## Features

- Browse available bots
- Add bots to your army (one per class)
- Release bots from your army
- Permanently discharge bots
- Filter bots by class
- Sort bots by health, damage, or armor
- View bot details

## Setup

1. Clone the repository
2. Run `npm install`
3. Start JSON server: `json-server --watch db.json --port 8001`
4. Start React app: `npm start`

## Technologies

- React
- JSON Server
- CSS3
